Sherlock - Anomaly Detector

Introduction

Sherlock is an anomaly detection service built on top of Druid. It leverages EGADS (Extensible Generic Anomaly Detection System) to detect anomalies in time-series data. Users can schedule jobs on an hourly, daily, weekly, or monthly basis, view anomaly reports from Sherlock's interface, or receive them via email or slack.

Contribute

Flash Query

Flash Query is a utility to perform instant Anomaly Detection on a Druid query. Users may enter a Druid query along with detection parameters in the Flash Query form. Sherlock detects anomalies in the time-series and displays them in a line chart and table visualization.

Job Scheduling

Sherlock provides Job Scheduling for a given Druid query to periodically monitor time-series. Users may create jobs via the New Job. Jobs List also provides a timeline chart for currently scheduled jobs.

Druid Cluster

User can add new Druid cluster using Add Cluster form. Druid Clusters shows the list of all druid clusters with their status and each cluster related information. User can update the cluster by selecting it from the list.

Reports

Sherlock provides Calendar-Heatmap visualization where periodically generated reports may be viewed. Reports show a table in which each row represents a time-series and associated groupby dimension in the Druid query with anomalies.
